FundMe Smart Contract

A crowdfunding smart contract built using the Foundry development framework. The contract allows users to fund with ETH (minimum $5 USD) and includes features like Chainlink Price Feeds integration, automated testing, and multi-chain deployment support.

Key features include ETH/USD price conversion using Chainlink oracles, minimum funding requirements, comprehensive testing suite, and gas-optimized operations. The project demonstrates expertise in smart contract development, testing practices, and Web3 integration.
